# Recipes

## Kasuya's Hybrid Method

[Source](https://www.youtube.com/watch?v=gC8K40kZ_6E)

| Brewer | Hario Switch |
|-----------|-------|
| In | 20g |
| Out | 300ml |

| Time | Action |
|------|--------|
| 0:00 | Close, pour until 50-60ml |
| 0:45 | Open, pour until 120ml |
| 1:30 | Pour until 200ml |
| 1:45 | Add water to kettle until 70C to 80C |
| 2:15 | Close, pour up to 300ml |
| 2:45 | Open |
| 3:30 | Finished (water drained completely) |
