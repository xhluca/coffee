# Recipes

## Kasuya's Hybrid Method for Hario Switch

[Source](https://www.youtube.com/watch?v=gC8K40kZ_6E)

In: 20g
Out: 300ml

1. [0:00] Close, pour until 50-60ml
2. [0:45] Open, pour until 120ml
3. [1:30] Pour until 200ml
4. [1:45] Add water to kettle until 70C to 80C
5. [2:15] Close, pour up to 300ml
6. [2:45] Open
7. [3:30] Finished (water drained completely)
