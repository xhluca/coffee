# Recipes

## Kasuya's Hybrid Method for Hario Switch

[Source](https://www.youtube.com/watch?v=gC8K40kZ_6E)

1. Close
2. [0:00] Pour until 50-60ml
3. [0:45] Open, pour until 120ml
4. [1:30] Pour until 200ml
5. [1:45] Add water to kettle until 70C to 80C
6. [2:15] Close, pour up to 300ml
7. [2:45] Open
8. [3:30] Finished (water drained completely)
